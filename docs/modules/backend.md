# Modulos backend

- auth: autenticacion, JWT, refresh tokens
- users: usuarios y roles
- parking: sesiones, cupos, reglas
- rates: tarifas y vigencias
- tickets: emision y estados
- devices: impresoras, lectores
- billing: caja y cierres
- sync: colas y reconciliacion
- audit: logs de cambios
- config: parametros globales
